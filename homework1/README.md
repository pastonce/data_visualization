### 网页打开

* 由于浏览器默认的安全策略，直接打开html文件时浏览器禁止读取本地csv文件，所以会无法显示
* 推荐使用`vscode`的`Live Server`插件或者`python`运行
* 运行前请务必在同目录下存放`iris.csv`文件
* 由于未适配不同的分辨率，所以请尽可能在较大分辨率的屏幕上查看
  
### 交互说明
  目录中的预览结果快照仅做结果的静态展示，实际运行网页可以进行以下交互：
  * 二维散点图：鼠标移动到图例上会突出显示该类全部样本；鼠标移动到样本点上会显示该样本详细数据
  * 平行坐标图：鼠标移动到图例上会突出显示该类全部样本；鼠标移动到样本线上会显示该样本详细数据
  * 雷达图：鼠标移动到样本多边形上会突出显示该类样本多边形；鼠标移动到样本多边形上会显示该区域统计数据；鼠标点击图例会交替显示该类样本多边形与具体的样本线
  * 箱线图：鼠标移动到图例上会突出显示该类样本的三个箱线；鼠标移动到样本箱线上会显示该箱线的详细统计数据
  * 三维散点图：支持鼠标控制视角的旋转与缩放
